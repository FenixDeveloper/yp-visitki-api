# Заглушка и АПИ к проекту ВИЗИТКИ

Файл `api.yml` — OpenAPI описание со стандартным сервером от [swaggerhub](https://app.swaggerhub.com/apis/FenixDeveloper/yp-business-card/1.0.0#/)
Директория `stub` — кастомная заглушка для самостоятельного деплоя

## Как настраивать
Файл `stub/api/visitki.js` — здесь объявлены все роуты и ответы на них, можно добавить еще таких файлов для других роутов. Все запросы будут префиксированы именем файла в виде `/visitki/users` и т.д.

Файл должен экспортировать объект, где каждый ключ это роут в формате:
METHOD:CODE:URI — например post:200:/users, здесь CODE это http-код ответа.
В качестве ответа отправляется JSON из значения присвоенного ключу. Если указать null, то будут отправлены только заголовки.

## Как запускать
Установить зависимости командой `yarn`
Для запуска `yarn start` или `./bin/www`

Принимает для настройки переменные окружения:
PORT — для http-сервера
BASE_PATH — для источника конфигов апи
